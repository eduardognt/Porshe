# Porsche Experience 🚗

Projeto desenvolvido com **React + Vite + JavaScript** com o objetivo de praticar e aprimorar meus conhecimentos em desenvolvimento Front-end.

A proposta foi criar uma landing page inspirada na identidade visual da Porsche, trabalhando componentes reutilizáveis, organização de código e principalmente manipulação de estado no React.

🔗 **Deploy:** https://porsche-teal.vercel.app

---

## 📸 Sobre o projeto

O projeto consiste em uma página de apresentação de um modelo Porsche, com foco em uma experiência visual moderna e interativa.

Durante o desenvolvimento, pratiquei conceitos importantes do React, como:

- Criação e reutilização de componentes;
- Organização de estrutura de projeto;
- Manipulação de estados com `useState`;
- Atualização dinâmica de elementos na interface;
- Componentização de uma página completa.

## 🎨 Funcionalidade principal

O principal desafio do projeto foi implementar a troca dinâmica das cores do veículo utilizando o hook **useState** do React.

Ao selecionar uma nova cor, o estado é atualizado e a interface altera a aparência do Porsche em tempo real, proporcionando uma experiência mais interativa para o usuário.

Exemplo do fluxo:

```
Usuário seleciona uma cor
          ↓
useState atualiza o estado
          ↓
Componente recebe a nova informação
          ↓
Imagem e elementos visuais são atualizados
```
